# Project Ujikom

Aplikasi berbasis web yang dibuat untuk memenuhi kebutuhan Uji Kompetensi (Ujikom) Rekayasa Perangkat Lunak (RPL). Sistem ini dirancang untuk membantu pengelolaan data secara efektif dan efisien melalui antarmuka yang mudah digunakan.

## 🚀 Fitur Utama

- Login dan Logout
- Dashboard Admin
- Manajemen Data
- CRUD (Create, Read, Update, Delete)
- Pencarian Data
- Laporan Data
- Responsive Design

## 🛠️ Teknologi yang Digunakan

- PHP
- MySQL
- Bootstrap
- JavaScript
- HTML5 & CSS3

## 📋 Persyaratan Sistem

- PHP 8.0+
- MySQL 5.7+
- XAMPP/Laragon
- Web Browser Modern
